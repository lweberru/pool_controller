# Advanced Pool Controller for Home Assistant

Eine intelligente Integration zur Steuerung von Whirlpools und Pools.
- Thermische Berechnung der Aufheizzeit basierend auf Wassermenge.
- Kalender-Integration für rechtzeitiges Aufheizen.
- Wasserqualitäts-Überwachung (PH, Chlor, TDS, Salz).
- PV-Überschuss-Logik & Frostschutz.
- Ruhezeiten-Management.
